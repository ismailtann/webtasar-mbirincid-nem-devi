• Html Nedir?

    • Hiper Metin İşaretleme Dili (HTML, İngilizce HyperText Markup Language kelimelerinin baş harflerinin kısaltılmasıdır) web sayfalarını oluşturmak için kullanılan standart metin işaretleme dilidir. Dilin son sürümü HTML5'tir. HTML, bir programlama dili olarak tanımlanamaz. Çünkü HTML kodlarıyla kendi başına çalışan bir program yazılamaz. Fakat bu dili yorumlayabilen programlar aracılığıyla çalışabilen programlar yazılabilir. Programlama dili denilememesinin nedeni tam olarak budur. Temel gereği, yazı, görüntü, video gibi değişik verileri ve bunları içeren sayfaları birbirine basitçe bağlamak, buna ek olarak, söz konusu sayfaların web tarayıcısı yazılımları tarafından düzgün olarak görüntülenmesi için gerekli kuralları belirlemektir. HTML kodunu web tarayıcıları okur, yorumlar ve görsel hale dönüştürürler, dolayısıyla aynı HTML kodunun farklı tarayıcılarda farklı sonuç vermesi olasıdır. CSS ve JavaScript ile beraber kullanıldığında HTML vasıtasıyla görsel ve dinamik web siteleri yaratılabilir.


  • Html En Çok Kullanılan Etiketler Nelerdir?

    • <h1> – <h6>: Başlık etiketleridir. <h1> en büyük başlığı temsil eder, <h6> ise en küçük başlığı. Web sayfasında içerik hiyerarşisi oluşturmak için kullanılır.

    <p>: Paragraf etiketidir. Metinleri paragraflar hâlinde düzenlemek için kullanılır ve tarayıcı otomatik olarak paragrafın başına ve sonuna boşluk bırakır.

    <a>: Link (bağlantı) etiketidir. Kullanıcıları başka sayfalara veya sitelere yönlendirmek için kullanılır. Örnek: <a href="https://example.com">Siteye Git</a>

    <img>: Resim etiketidir. Web sayfasına görsel eklemek için kullanılır. Önemli özellikleri src (resim kaynağı) ve alt (alternatif metin) özellikleridir.

    <ul> ve <ol>: Liste etiketleridir. <ul> sırasız liste (bullet point), <ol> ise sıralı liste (numaralı) oluşturur. İçinde <li> etiketi ile maddeler tanımlanır.

    <li>: Liste maddesi etiketidir. <ul> veya <ol> içinde kullanılır ve her bir liste öğesini temsil eder.

    <div>: Sayfa içinde blok oluşturmak için kullanılan genel bir kapsayıcı etiketidir. Stil vermek, içerikleri gruplamak ve sayfa düzeni oluşturmak için sıklıkla kullanılır.

    <span>: Küçük metin veya içerik parçalarını stil vermek için kullanılan satır içi etikettir. Blok oluşturmaz, sadece belirli bir kısmı etkilemek için kullanılır.

    <strong> ve <b>: Metni kalın yapmak için kullanılır. <strong> aynı zamanda vurgulu anlam taşır, <b> ise sadece görsel olarak kalın yapar.

    <em> ve <i>: Metni italik yapmak için kullanılır. <em> vurgulu anlam taşır, <i> ise sadece görsel olarak italik yapar.

    <form>, <input> ve <button>: Kullanıcıdan veri almak için kullanılan etiketlerdir. Formlar web sayfasında veri toplamak için kullanılır, <input> veri giriş alanı, <button> ise buton oluşturur.


  • Html İle Web Sitesi Nasıl Yapılır?

    • HTML ile web sitesi oluşturmak için öncelikle sayfanın yapısını belirleyen HTML kodları yazılır. Başlıklar, paragraflar, listeler, linkler ve resimler gibi içerikler bu kodlarla sayfaya eklenir. Sayfayı görsel olarak düzenlemek için CSS, interaktif özellikler için JavaScript kullanılır. HTML, CSS ve JavaScript birlikte çalışarak kullanıcıların görebileceği ve etkileşimde bulunabileceği bir web sayfası ortaya çıkarır.
